Realizations

I use my Raspberry Pi 4 to self-host a number of different software applications.

1. My Discord bot

My Discord bot written in Python is hosted on this homelab so that it runs 24/7 and everyone can always use it. You can read more about my Discord bot here.

2. Wiki website

I have created a wiki website using the DokuWiki tool. This wiki contains documentation about Racu (my Discord bot) and is hosted on my Raspberry Pi using an nginx reverse proxy with Docker. The wiki page is accessible via this link: https://wiki.wlinator.org/

3. Git server

I have a self-hosted Git server where the source code I write is preserved, next to Github. This is mostly for software preservation but self-hosting Forgejo was a great way for me to learn the ins-and-outs of Git. https://git.wlinator.org/

4. Various smaller pieces of software

Besides my bot, wiki page and git server, I also host smaller things like Pi-Hole, adminer, thelounge.chat, …
